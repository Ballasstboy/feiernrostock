# Feiern Rostock - Nightlife Discovery Platform

A modern, responsive web application for discovering the best nightlife in Rostock, Germany. Find clubs, browse events, and plan your perfect night out with an intuitive interface and comprehensive event information.

## 🌟 Features

### Core Features
- **Interactive Club Cards**: Browse all clubs in Rostock with detailed information including logos, descriptions, and contact details
- **2-Week Event Schedules**: Click any club to view comprehensive event listings with dates, times, prices, and event details
- **Event Comparison**: Compare upcoming events across all clubs to decide where to party
- **Genre Filtering**: Filter clubs and events by music genre (Techno, House, Hip-Hop, Charts, etc.)
- **Dark Theme**: Nightlife-inspired dark mode design optimized for evening browsing

### Advanced Features
- **Interactive Map View**: Explore club locations on an interactive map with custom markers
- **Mobile-First Design**: Fully responsive interface optimized for mobile devices
- **Event Details**: Complete event information including DJs, dress codes, age limits, and pricing
- **Real-time Filtering**: Dynamic filtering system that updates results instantly

## 🚀 Technology Stack

- **Frontend**: React 18 with TypeScript
- **Maps**: Leaflet with React-Leaflet integration
- **Icons**: Lucide React for modern iconography
- **Styling**: Custom CSS with CSS Variables for theming
- **Date Handling**: date-fns for robust date manipulation
- **Build Tool**: Create React App with TypeScript template

## 🛠️ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd feiernrostock
   ```

2. **Navigate to the project directory**
   ```bash
   cd rostock-nightlife
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm start
   ```

5. **Open your browser**
   Navigate to `http://localhost:3000` to view the application

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App (irreversible)
